Inicio

Objetivo General:

Crear un OVI objeto virtual de información que nos permita representar la tematica propuesta en el curso de diseño de sistemas elegida por cada integrante del grupo, identificando y aplicando sus funcionalidades como guardar el historial de cambios sobre el codigo de un proyecto, permitir a varias personas contribuir simultaneamente acerca del proyecto, entre otras.

Objetivos especificos:

* Establecer la metodologia y estructura basica con la que se fundamenta nuestro objeto virtual de información, haciendo uso del control de versiones que nos permite trabajar de una manera mas simple y optima cuando estamos desarrollando software.

* Dar a conocer información acerca de lo que es GIT Y GITHUB y la configuracion inicial de cada uno.

* Guardar cambios realizados dentro del codigo de un proyecto, teniendo en cuenta los requerimientos con los que se fundamenta la herramienta GIT Y GITHUB.

Temática

En la presente sección damos la bienvenida a este OVI del curso de diseño de sitios web, en el cual abordaremos el tema correspondiente a GIT Y GITHUB como, conceptos y configuraciones basicas.
