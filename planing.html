GIT Y GITHUB

Objetivo General

Crear un OVI objeto virtual de información que nos permita representar la tematica propuesta en el curso de diseño de sistemas elegida por cada integrante del grupo, identificando y aplicando sus funcionalidades como guardar el historial de cambios sobre el codigo de un proyecto, permitir a varias personas contribuir simultaneamente acerca del proyecto, entre otras.

objetivos especificos

  • Establecer la metodologia y estructura basica con la que se fundamenta nuestro objeto virtual de información, haciendo uso del control de versiones como es git que nos permite trabajar en equipo de una manera mas simple y optima cuando estamos desarrollando software.
  • Alojar proyectos en github utilizando el sistema de control de versiones de git de forma publica aunque se puede hacer de forma privada creando una cuenta de pago.
  • Expresar de manera creativa y organizada nuestro OVI ya que nos permite visualizar informacion relacionada a la tematica elegida, Git y Github en el desarrollo de la presente actividad.
